# SCArchive2
https://scarchive.kr
스타크래프트 데이터 저장 프로그램 SCArchive입니다.

유즈맵에서 플레이한 데이터를 온라인 서버에 저장하여 어디서든지 불러 올 수 있습니다.

Zip 비밀번호는 scarchive입니다.
